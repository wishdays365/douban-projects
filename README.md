**时间**：大一（2025年12月 — 2026年1月）  
**技术栈**：Python + Flask + BeautifulSoup + Matplotlib + ECharts

## 功能

- 爬取豆瓣电影 Top250 数据
- 演员合作关系分析
- 电影类型分布统计
- 排名与评分可视化
- 电影地图展示（ECharts）
- Web 管理面板（Flask + Jinja2）

## 运行

```bash
pip install -r requirements.txt
python app.py
# 访问 http://127.0.0.1:5000
```
