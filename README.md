# 豆瓣 Top250 电影数据分析系统

**一站式豆瓣电影数据自动爬取、检索与可视化分析平台**  
大一项目 | 2025 年 12 月 — 2026 年 1 月

---

## 项目概览

这是一个完整的 **Web 全栈项目**，能够自动爬取豆瓣电影 Top250 数据，并将分析结果以精美的 Flask Web 页面呈现。整个系统实现了从数据采集、清洗、存储到多维度可视化分析的完整闭环。

> 首页展示核心统计（平均评分、年份跨度等），导航栏可快速进入检索、排名、类型分析、演员统计、国家分布等功能页面。

---

## 主要功能

| 功能 | 描述 |
|------|------|
| 🕷️ **自动爬虫** | 异步并发爬取豆瓣 Top250，集成 Cookie 反反爬，支持断点续爬 |
| 🔍 **多维度检索** | 按标题 / 导演 / 演员 / 类型 / 国家检索，可筛选评分和年份 |
| 🏆 **电影排行** | 高分榜 / 热门榜 / 最新榜三大维度，支持自定义展示数量 |
| 🎭 **类型分析** | ECharts 可视化展示各类型电影数量分布 |
| 👥 **演员统计** | 统计出镜次数最多的演员，发现高分电影常客 |
| 🌍 **国家分布** | 世界地图可视化各国 / 地区电影数量 |
| 📊 **Excel 报告** | 自动生成带格式的统计报告（openpyxl） |
| ⏱️ **自动维护** | 每 3 小时自动检测数据新鲜度，过期自动重新爬取 |
| 📝 **日志系统** | 完整的日志记录，支持文件轮转（10MB / 5 备份） |
| 🚀 **WSGI 部署** | 支持通过 Gunicorn / uWSGI 生产环境部署 |

## 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动服务
python app.py

# 3. 打开浏览器
# http://127.0.0.1:5000
```

首次启动后系统会自动检测数据状态，若无数据则自动触发爬取（约 1-2 分钟完成 Top250 采集）。

## 页面功能

| 页面 | 路由 | 功能 |
|------|------|------|
| 首页 | `/` | 数据状态实时显示 + 核心统计卡片（平均评分、年份跨度等） |
| 数据状态 | `/crawl` | 爬取进度、数据新鲜度、手动触发爬取 |
| 电影检索 | `/search` | 多维度搜索 + 评分 / 年份筛选 |
| 电影排行 | `/rank` | 高分 / 热门 / 最新三榜，可调展示数量 |
| 类型分析 | `/genre-analysis` | ECharts 柱状图展示各类型数量分布 |
| 演员统计 | `/actor-analysis` | ECharts 展示出镜次数 Top N 演员 |
| 国家分布 | `/movie-map` | ECharts 世界地图展示各国电影数量 |
